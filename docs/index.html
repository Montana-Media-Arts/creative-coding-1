    University of Montana, School of Media Arts

    This is the website for the online section of Creative Coding 1 offered by The School of Media Arts, University of Montana.

    This course is designed to teach you code so that it can become a tool for your art and creativity.

    Utilizing p5.js, this course teaches concepts from computer science and programming. These will occur through visual and audio sketches that run in web browsers.

    The syllabus contains more information about the course, as well as student expectations.

    This course is taught by Michael Musick.
